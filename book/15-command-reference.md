# Complete Command Reference

This chapter provides a comprehensive reference of VoiceOver commands, organized by category for quick lookup.

## General Commands

| Action | Command |
|--------|---------|
| Toggle VoiceOver on/off | **Command + F5** |
| Open VoiceOver Utility | **VO + F8** |
| Open VoiceOver Tutorial | **VO + Command + F8** |
| Open VoiceOver Help menu | **VO + H** |
| Open Commands Help | **VO + H + H** |
| Keyboard Help (practice) | **VO + K** |
| Lock/unlock VO keys | **VO + ; (semicolon)** |
| Stop speech | **Control** |
| Repeat last phrase | **VO + Z** |
| Copy last phrase to clipboard | **VO + Shift + C** |
| Save last phrase to file | **VO + Shift + Z** |

## Quick Reference Card

### Most Essential Commands

| I Want To... | Press... |
|--------------|----------|
| Turn VoiceOver on/off | **Command + F5** |
| Move to next item | **VO + Right Arrow** |
| Move to previous item | **VO + Left Arrow** |
| Activate item | **VO + Space** |
| Enter a group | **VO + Shift + Down Arrow** |
| Exit a group | **VO + Shift + Up Arrow** |
| Read continuously | **VO + A** |
| Stop reading | **Control** |
| Open Rotor | **VO + U** |
| Go to Menu Bar | **VO + M** |
| Go to Dock | **VO + D** |
| Next heading (web) | **VO + Command + H** |
| Next link (web) | **VO + Command + L** |
| Get Help | **VO + H** |

### Remember

- **VO** = Control + Option (or Caps Lock if enabled)
- Add **Shift** to go to previous instead of next
- Add **Command** to navigate to specific element types
- Press commands **twice** for additional functions
- **Escape** usually cancels or exits
